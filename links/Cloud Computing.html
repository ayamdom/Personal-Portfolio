Cloud Computing

I got interest on how computing resources like servers, storage, and software access over the internet ("the cloud").


IaaS (Infrastructure as a Service) provides IT infrastructure like servers, and storage over the internet.


PaaS (Platform as a Service) provides platform to build, deploy, and manage applications without the complexity of managing the underlying infrastructure.


SaaS(Software as a Service) delivers software applications over the internet, and access the software through a web browser, or mobile app.


FaaS(Function as a Service) relatively new cloud service model that provides solutions to build applications as simple, event-triggered functions without managing or scaling any infrastructure.
