Quality Assurance Test

I have keen interest on different kinds of testing to ensure that the software is reliable and quality before it proceeds to deployment.


Functional Testing verifying that the software application behaves as expected based on the requirements.

Non-Functional Testing verifies how well the software performs, rather than what it does.

Manual Testing performed by human testers without the aid of automated tools.

Automated Testing using specialized software tools to execute pre-written test scripts and compare the actual results with the expected results.
