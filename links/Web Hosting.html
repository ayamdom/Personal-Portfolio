Web Hosting

My curiosity on how to make a sites accessible via the World Wide Web gets deeper as I have already experience deploying static and dynamic site over the internet.


Web Servers are powerful computers that host website's files specifically the properly configured files making it accessible over the internet.

Storage hosting site providers offer space on their servers to store website's files, the amount of storage will depends on the size and complexity of the website.

Bandwidth the traffic flow to and from a website, it's the amount of data transferred when a user access a website.

Uptime the amount of time that a web server is running and accessible to users, high uptime like 99.9% means a user can access the website.

Domain Name is the address of a website on the internet and its the address that the people type into their web browser to find specific website.
